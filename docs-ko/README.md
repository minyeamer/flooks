# FLooks 한국어 문서 안내

이 경로는 사람 사용자가 읽고 바로 판단하거나 행동에 옮기기 위한 한국어 문서를 둔다.

언어 정책은 다음과 같다.

- `docs/` 는 프로젝트 정본 문서 경로이며 영어만 사용한다.
- `docs-ko/` 는 사용자용 한국어 문서 경로이며 한국어만 사용한다.
- AI 에이전트, 코드 생성 도구, 저장소 규칙은 `docs/` 를 우선 참조한다.
- 한국어 문서는 제품 방향 설명, 실행 판단, 보고용 요약처럼 사용자가 직접 소비하는 내용을 위해 유지한다.

현재 한국어 문서:

- `docs-ko/architecture/platform-dossier.md`
- `docs-ko/architecture/overview.md`
- `docs-ko/architecture/query-spec.md`
